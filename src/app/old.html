WEB Projects
Well known Supermarket;-): AI Development
2024-25

Tasked with converting elearning storyboards (Word docs) into a course files for popular elearning platform (Adapt) I prototyped a conversion tool that uses AI LLM models (OpenAI API & Llama open source), Python API backend and an Angular frontend to analyse the content, discover the intent and obvious elements required for each screen then map the text, image, media and interactive elements to the core Adapt models. This would, in one click, generate a working Adapt course in seconds where it can be imported into the Adapt Authoring tool for additional editing.

Another vendor also created a storyboard generator that created basic content that I also adapted my tool to work with. Next steps are to convert the tool into a cmd line version.
