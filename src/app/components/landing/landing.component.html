Selected Projects

AI-Powered Courseware Generator

Developed an innovative eLearning platform that automatically converts storyboards into working courses in the Adapt framework. Using LLM models (both API and open source), custom agents, Python, and Angular, the system analyzes content to detect visual elements, learning intentions, outcomes, and interactions. The AI then maps these elements to the appropriate Adapt framework components, creating fully functional eLearning courses with a single click.

Veterans Support Platform

Developed three interconnected websites for Australian Catholic University to support military veterans transitioning to university life. As the sole developer, I designed the sites in WebFlow, exported them as HTML, and integrated advanced features using React components, including a sophisticated referencing and citations tool. The project involved converting complex data from university-provided spreadsheets into JSON to drive dynamic component functionality, helping veterans navigate their educational journey.

Lesson Planner Platform

Built an educational platform using Python for curriculum parsing and alignment with the Victorian State Curriculum. The system automatically analyzes and maps content to curriculum requirements, ensuring educational standards are met. Developed with an Angular frontend and Lumen API backend, the project included a WordPress marketplace integration for content distribution. The platform helps teachers efficiently plan and align their lessons with curriculum requirements.

The series has become an essential resource for new researchers and ongoing staff training.
